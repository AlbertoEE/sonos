# Atributos del Cantor - Referencia Completa

## Los 6 Atributos

| Atributo | En Combate | En Rol |
|----------|-----------|--------|
| **Ritmo** | Potencia física, velocidad, impacto directo | Fuerza bruta, esfuerzo físico, resistencia |
| **Melodía** | Precisión, creatividad, expresividad musical | Sensibilidad artística, percepción fina, intuición creativa |
| **Armonía** | Control, protección, calma, equilibrio | Empatía, autocontrol, liderazgo sereno, cuidado |
| **Disonancia** | Caos, ruptura, agresividad, efectos oscuros | Crueldad, temeridad, pulsiones destructivas |
| **Síncopa** | Trucos, engaño, manipulación | Engaño social, astucia, lectura de intenciones |
| **Tempo** | Rapidez de reacción, iniciativa de grupo | Reflejos mentales, rapidez de decisión |

## Rango de Valores
- **Mínimo:** -5
- **Máximo:** +5
- **Solo se modifican:** Con equipamiento, rudimentos o fundamentos

## Afinidad del Instrumento Vital

El Instrumento Vital tiene una **afinidad** con cada atributo (S, A, B, C, D):

| Nivel | % del Atributo | Mejora Plana |
|-------|---|---|
| **S** | 200% | +4 |
| **A** | 200% | — |
| **B** | 100% | +2 |
| **C** | 100% | — |
| **D** | 50% | — |

### Cálculo del Atributo Total del IV
`Atributo Base × % + Mejora Plana = Atributo Total`

Este valor se usa para daño y otros efectos del IV.
