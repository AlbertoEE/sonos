# Mejoras del Instrumento Vital - Guía de Progresión

## Sistema de Mejoras

Las mejoras del IV dependen del **atributo dominante** del instrumento (el que tiene más puntos).

Cada mejora aplica un **bonificador** repartido según las afinidades.

## Recursos Derivados de Mejoras

### Vitalidad (PV)

Se calcula con **Ritmo como base principal**:

| Atributo | Fórmula |
|----------|---------|
| **Ritmo dominante** | Base (21) + 10 × [número de mejoras] |
| **Cualquier otro** | Base (16-18) + bonus según mejora |

**Ejemplo:** IV con Ritmo S obtiene +20 PV por cada mejora.

### Silencio (Defensa)

Cada mejora **suma dados extra**:

| Atributo | Dado Base | Suma por Mejora |
|----------|-----------|---|
| **Ritmo** | 3d4 | +3 |
| **Melodía** | 1d10 | +1 |
| **Armonía** | 1d10 | +1 |
| **Disonancia** | 2d6 | +2 |
| **Síncopa** | 2d6 | +2 |
| **Tempo** | 1d10 | +1 |

**Cómo se aplica:** Tiras todos los dados cada vez.

### Resonancia (Energía)

El máximo **no aumenta con mejoras directas**, pero:
- Fundamentos y Rudimentos pueden afectar recuperación
- Acción "Afinar" reduce Tensión (-1) por 2 PA

| Efecto | Recuperación |
|--------|---|
| Nuevo Pulso | +1 Resonancia |
| Crítico en Silencio | +1 Resonancia |
| Ejecutar enemigo | +1 Resonancia |

## Equipamiento Especial

El IV puede tener equipamiento que:

| Tipo | Efecto |
|------|--------|
| **Mejora de Dado** | Cambia 1d8 → 1d10, 1d12, etc. |
| **Ampliación de Alcance** | Extiende rango máximo |
| **Resistencias** | Reduce ciertos tipos de daño |
| **Fundamentos Especiales** | Desbloquea habilidades nuevas |

## Estrategia de Mejoras

### Camino Defensivo (Armonía/Ritmo)
- ✓ Más PV
- ✓ Más Silencio
- ✓ Mejor tanque
- ✗ Menos daño puro

**Recomendado para:** Defensar aliados, absorber daño

### Camino Ofensivo (Ritmo/Disonancia/Melodía)
- ✓ Más daño
- ✓ Mejor precisión
- ✓ Críticos más fuertes
- ✗ Menos defensa

**Recomendado para:** DPS, eliminación rápida

### Camino de Utilidad (Síncopa/Tempo)
- ✓ Mejor iniciativa
- ✓ Más trucos disponibles
- ✓ Control del campo
- ✗ Menos daño bruto

**Recomendado para:** Control, Soporte, Utilidad

---

## Recuerda

- **Mejoras son permanentes** (a menos que pierdas equipamiento)
- **Recalcula Vitalidad y Silencio** cuando cambies mejoras
- **Las mejoras se apilan:** +1 después +1 = bonificador doble
- **Comparables con niveles en otros sistemas**
