# Sonos — Contexto General del Proyecto

Este directorio contiene **Sonos**, un sistema TTRPG original que combina mecánicas de Draw Steel, D&D 5e, Pathfinder 2e y Stars Without Numbers, ambientado en un planeta donde la música es el fundamento de toda magia y poder.

## Estructura del Vault

```
1. Empieza Aquí/     → Intro al sistema y al planeta
2. Jugadores/        → Fichas de los PJs (Ángeles, Cae, Juan, Raquel, Rodri)
3. Recaps/           → Resúmenes de sesiones pasadas
4. Otros/rules/      → Reglas completas del sistema
4. Otros/lore/       → Lore del mundo (lugares, facciones, NPCs)
4. Otros/items/      → Objetos, consumibles, recetas
campaña/sesiones/    → Preparación y notas de cada sesión
cheatsheets/         → Referencias rápidas para la mesa
AI Prompts/          → Prompts especializados para Claude
```

## Agentes Especializados

- **Reglas de Sonos** → trabaja desde `4. Otros/rules/` — consulta reglas del sistema
- **Campaña de Sonos** → trabaja desde `campaña/` — conoce la historia, NPCs y estado actual

## Convenciones del Sistema

- Los dados van de d2 a d100, organizados en **Categorías de dado**
- El sistema de tiradas usa **2d10** con 3 tiers (filosofía Draw Steel)
- Cada personaje tiene un **Instrumento Vital (IV)** con atributos propios
- Recursos principales: **Vitalidad, Silencio, Resonancia, Auralium**
- Combate: jugadores primero → enemigos después. DM acumula **Malicia** (1/turno)
- **Tensión** sube el riesgo y el daño durante el combate (escala 0-10)
- Moneda: **Alts (Ats)**
