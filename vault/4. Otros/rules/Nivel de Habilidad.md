# Nivel de Habilidad

Las habilidades de un Cantor tienen **varios niveles**. Todas comienzan en **0**.

| Nivel | Significado narrativo | Mod |
|-------|------------------------|-----|
| 0-19  | No entrenado           | -1  |
| 20-39 | Iniciado               | 0   |
| 40-59 | Competente             | 1   |
| 60-79 | Experto                | 2   |
| 80-99 | Maestro                | 3   |
| 100   | Eminencia              | 4   |

Para progresar hace falta llegar al menos a **Iniciado**; ver [[Progresión de Habilidades]].

Ver: [[Habilidades Sonos]], [[Checks de Habilidades]].
