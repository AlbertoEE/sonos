# Progresión de Habilidades

## Aprender una Habilidad Nueva

Para progresar en una habilidad primero necesitas el rango **Iniciado**:

- **Con un Profesor** (rango "Experto"): cada sesión tira **2d6+3**. Con el profesor no se puede superar 20; al llegar a 20 empieza el camino autodidacta.
- **Por tu cuenta**: necesitas materiales de práctica. Una vez por **downtime** tira **1d6** y súmalo a la habilidad. Dependiendo el Resultado:
	- **1**: pierdes la mitad del progreso.
	- **2, 3, 4, 5, 6**: Súmalo a tu progreso actual  
	- **6**: duplica tu progreso o suma 6.

## Subir de Nivel una Habilidad

Existen diferentes maneras de subir de nivel una habilidad y todas ellas ocurren en partida al hacer un [[Checks de Habilidades|skill check]]:

| Tier    | Resultado                      |
| ------- | ------------------------------ |
| Pifia   | Suma 3d4 Puntos a tu Habilidad |
| Tier 1  | Suma 3 Puntos a tu Habilidad   |
| Tier 2  | Suma 2 Punto de Habilidad      |
| Tier 3  | Suma 1 Punto de Habilidad      |
| Crítico | Suma 0 Puntos de Habilidad     |

Ver: [[Habilidades Sonos]], [[Nivel de Habilidad]], [[Checks de Habilidades]].
