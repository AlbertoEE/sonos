# Descripción
Sabroso fluido viscoso, denso y dulce. Obtenido de la cocción lenta de cáscaras de [[Smrug]].

# Materiales
- 2 Cáscaras de [[Smrug]]
- Agua

# Preparación
Tiempo de preparación: 4 horas

| Tier    | Resultado                               |
| ------- | --------------------------------------- |
| Pifia   | No ocurre, nada, pierdes los materiales |
| 1       | Obtén [[Caramelo de Smrug]] 1          |
| 2       | Obtén [[Caramelo de Smrug]] 2          |
| 3       | Obtén [[Caramelo de Smrug]] 3          |
| Crítico | Obtén [[Membrillo de Smrug]]           |
