# Descripción
Un frasco con un aroma indeterminado que cambia la [[Actitud]] de un NPC haciéndole más amigable. Esta poción no puede ser usada con un enemigo que ya ha iniciado el combate. ^0a0b2d

# Materiales
- [[Frasco]]
- [[Flor de Lis]]
- 100 Ats pulverizados

# Preparación
Tiempo de preparación: 30 minutos

| Tier    | Resultado                                     |
| ------- | --------------------------------------------- |
| Pifia   | No ocurre, nada, pierdes los materiales       |
| 1       | Obtén [[Frasco de Fragancia Reconfortante]] 1 |
| 2       | Obtén [[Frasco de Fragancia Reconfortante]] 2 |
| 3       | Obtén [[Frasco de Fragancia Reconfortante]] 3 |
| Crítico | Obtén [[Frasco de Fragancia Embelesadora]] 3 |
