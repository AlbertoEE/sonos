# Sonos - Vistas Dataview

## Recetas

```dataview
TABLE file.name AS "Nombre", precio AS "Precio", gremio AS "Gremio"
FROM "Sonos/4. Otros/items"
WHERE tipo = "receta"
SORT gremio ASC, file.name ASC
```

## Materiales


## Consumibles

## Tienda

```dataview
TABLE file.name AS "Nombre", precio AS "Precio", gremio AS "Gremio", tipo AS "Tipo"
FROM "Sonos/4. Otros/items"
WHERE tipo != "material" AND gremio = "cocina"
SORT gremio ASC, tipo ASC, file.name DESC, precio DESC
```
