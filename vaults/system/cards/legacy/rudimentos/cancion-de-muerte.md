# Canción de Muerte

## Base
Lanza [1d DA]. El resultado determina la **Resonancia** que todos los objetivos en el campo de batalla (incluido tú) deben gastar para cancelar este efecto. Si no pagan, mueren. Si alguien muere, este Rudimento se desactiva.
