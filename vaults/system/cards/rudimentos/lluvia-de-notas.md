# Lluvia de Notas

## Base
Crea un área de 3x3 casillas en un rango de 3 casillas. Al principio de cada ronda, cualquier objetivo en el área recibe [AT] de *daño verdadero*. El efecto dura X rondas.

## Evolucionado
El área se vuelve de 4x4 casillas y el daño solo aplica a enemigos.
