# Baluarte

## Base
Coloca una estructura en una casilla adyacente vacía con **Vitalidad** igual a tu [DI]. Si un aliado se encuentra directamente detrás de ella, la estructura intercepta automáticamente el daño dirigido a ese aliado. Máximo de estructuras activas [AT/3].
