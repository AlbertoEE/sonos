# Baluarte

## Base
Tira [1DI+AT]. Coloca una estructura en una casilla adyacente vacía con **Vitalidad** igual al resultado. Si un aliado se encuentra directamente detrás de ella, la estructura intercepta automáticamente el daño dirigido a ese aliado. Máximo de estructuras activas: máx(1, AT÷3).
