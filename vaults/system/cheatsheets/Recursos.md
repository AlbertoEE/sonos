# Recursos del Cantor - Referencia Completa

## Vitalidad (PV)

Determina **cuánto daño puedes recibir** antes de caer.

### Valores Máximos por Atributo Dominante del IV

| Atributo | Base | Progresión por Mejora |
|----------|------|---|
| **Ritmo** | 21 | +10 |
| **Melodía** | 16 | +5 |
| **Armonía** | 16 | +5 |
| **Disonancia** | 18 | +7 |
| **Síncopa** | 18 | +7 |
| **Tempo** | 16 | +5 |

### A 0 de Vitalidad
- **Forma animal:** Retornas a forma original con 1 PV
- **Forma original:** Quedas inconsciente
- **Inconsciente + Daño = 50% Vitalidad máxima:** Mueres

## Silencio (Defensa)

Tu capacidad de **reducir daño entrante** cada turno. Se tira cada vez que recibes daño.

| Atributo | Dado Base | Mejora por Nivel |
|----------|-----------|---|
| **Ritmo** | 3d4 | +3 |
| **Melodía** | 1d10 | +1 |
| **Armonía** | 1d10 | +1 |
| **Disonancia** | 2d6 | +2 |
| **Síncopa** | 2d6 | +2 |
| **Tempo** | 1d10 | +1 |

### Resultados Especiales en Silencio

| Resultado | Efecto |
|-----------|--------|
| **Pífia** | Daño entrante = Daño Verdadero (penetra defensa) |
| **Crítico** | Bloquea TODO el daño + Recupera 1 Resonancia |
| **Parry** | Si daño (pre-mods) = Silencio (pre-mods): Devuelves todo como daño verdadero |

### Mejoras en Combate
- **Acción "Defenderse"** (2 PA): Duplica Silencio en la siguiente ronda
- **Acción "On Garde"** (2 PA): Si Silencio cubre todo, devuelves todo como daño verdadero

## Resonancia (Energía para Rudimentos)

Recurso para **ejecutar rudimentos**. Comienza el combate con LA MITAD de máximo.

| Atributo | Máximo | Inicio Combate |
|----------|--------|---|
| **Ritmo** | 4 | 2 |
| **Melodía** | 6 | 3 |
| **Armonía** | 6 | 3 |
| **Disonancia** | 4 | 2 |
| **Síncopa** | 4 | 2 |
| **Tempo** | 6 | 3 |

### Cómo Recuperar Resonancia

| Acción | Recuperación |
|--------|--------------|
| Primer Pulso de tu turno | +1 |
| Pulso posterior que inflige daño | +1 |
| Crítico en Silencio | +1 |
| Ejecutar a un enemigo | +1 |

### Efectos de Tensión
Ver [[Tensión]] para efectos adicionales sobre Resonancia.

## Auralium (Catalizador)

Recurso para **regenerar Vitalidad** rápidamente en combate.

| Característica | Valor |
|---|---|
| **Cargas iniciales** | 3 |
| **Máximo** | 3 |
| **PV recuperados** | 50% de Vitalidad máxima (forma animal) |
| **Costo en combate** | 2 PA (Acción "Súplica") |
| **Recarga** | Descanso completo o "Nana de la Súplica" |

---

## Resumen Rápido por Situación

### Inicio de Combate
- Vitalidad: 100%
- Resonancia: 50% del máximo
- Auralium: 3 cargas (si descansaste bien)
- Silencio: Tiras nuevo cada turno

### Turno Típico
1. Usas Pulsos (recuperas Resonancia)
2. Si usas Silencio y sacas Crítico: +1 Resonancia
3. Si necesitas curación: Usa Súplica (2 PA)

### Después del Combate
- Resonancia: Se drena
- Auralium: Se recupera con descanso
- Vitalidad: Solo se recupera fuera de combate o con magia
