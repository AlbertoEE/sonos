# Alpha 0.2.1 — Ritmo de Auralita Refinada

## Intent

Definir la Auralita Refinada como el principal recurso de progreso de personaje y dar al DM una fórmula sencilla para ajustar cuánta Auralita Refinada entregar por sesión según la duración deseada de la campaña.

## Changes

- Added a `Ritmo de entrega de Auralita Refinada` section to `Mejoras del Instrumento Vital`.
- Defined that Sonos does not use levels: each Cantor invests Auralita Refinada in their Instrumento Vital.
- Established the recommended reward as 5 Auralita Refinada per Cantor at the end of each session.
- Clarified that Auralita Refinada is tangible and can be shared between players.
- Added the formula: `Auralita por Cantor por sesión = Coste objetivo de progreso / Número de sesiones deseadas`.
- Added optional end-of-session extra rewards for table highlights such as funniest player, most epic moment, or best roll.

## Design Notes

The default of 5 Auralita Refinada per Cantor per session keeps progression simple and arcade-like, reducing DM overhead. The formula lets each DM tune the progression speed without changing the underlying improvement costs.
