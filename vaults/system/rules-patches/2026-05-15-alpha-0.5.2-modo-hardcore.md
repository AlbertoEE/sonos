# Alpha 0.5.2 — Modo Hardcore

## Intent

Añadir el Modo Hardcore como modo alternativo y clásico para la gestión de muerte: cuando un personaje muere, el combate no se reinicia, la muerte es definitiva y el jugador debe crear otro personaje.

## Changes

- Aclarado que Sonos asume por defecto el enfoque **Arcade**, gestionado mediante las reglas de Resintonía.
- Añadido **Modo Hardcore** como modo alternativo y clásico.
- Definido que en Modo Hardcore, cuando un personaje muere, el combate no se reinicia.
- Definido que en Modo Hardcore la muerte del personaje es definitiva y el jugador debe crear otro personaje.

## Design Notes

Este patch no define todavía un procedimiento detallado de reinicio para el Modo Arcade; solo diferencia el modo alternativo Hardcore frente al enfoque Arcade por defecto.
