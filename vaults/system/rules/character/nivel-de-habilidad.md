# Nivel de Habilidad
Las habilidades de un cantor tienen varios niveles. Todas comienzan al 0. 

| Nivel | Significado narrativo | Mod |
| ----- | --------------------- |-----|
| 0-19  | No entrenado          | -1  |
| 20-39 | Iniciado              |  0  |
| 40-59 | Competente            |  1  |
| 60-79 | Experto               |  2  |
| 80-99 | Maestro               |  3  |
| 100   | Eminencia             |  4  |

Al crear un Cantor se recomienda comenzar con máximo 4 hablidades nivel "Competente".
