# Bajada de Tensión
| Acción                                | Efecto      |
|---------------------------------------|-------------|
| Un enemigo lanza una pifia            | −1          |
| Un jugador consigue un crítico        | −1          |
| Afinar (Acción Principal)             | −1          |
| Resolver (acción de grupo, 1/combate) | Tensión a 0 |
